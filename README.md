# bookmark2pdf
add bookmark to pdf with PyPDF2.

## use
```
pip install PyPDF2
python generate_bookmarks.py --input=data_structures_algorithms_in_go.pdf --txt=table_of_content.txt --output=data structures & algorithms in go(with bookmark).pdf
```
